# 📻 RetroStream Internet Radio

RetroStream е красиво, винтидж-стилизирано приложение за слушане на интернет радио. Проектът е изграден първоначално като уеб приложение (PWA) и след това е напълно оптимизиран и компилиран като **нативно Android приложение** чрез Capacitor.

## ✨ Ключови функционалности

*   **🎧 Хиляди станции:** Достъп до световната база данни на радиостанции благодарение на интеграцията с *RadioBrowser API*.
*   **🔋 Хардуерна оптимизация (GPU):** Интерфейсът и плъзгащият се текст (Marquee) използват CSS `transform` анимации, изчислявани директно от видеокартата на телефона, запазвайки батерията.
*   **📱 Нативно фоново аудио (ExoPlayer):** На Android, приложението използва хардуерно аудио декодиране чрез `ExoMedia` и се представя пред сървърите (като Icecast) като `VLC/3.0.18`, за да предотврати прекъсвания на стрийма.
*   **⏸️ Интелигентен фонов режим:** Всички визуални броячи и ефекти спират да консумират CPU ресурс, веднага щом телефонът бъде заключен (чрез `visibilitychange` API).
*   **⭐ Любими станции:** Запазване на любими станции с възможност за Експорт/Импорт (Export/Import) във формат `.json`.
*   **🎨 Ретро дизайн:** Изчистен, тъмен дизайн с тактилни бутони и кехлибарени (Amber) акценти.

## 🛠️ Технологичен стек

*   **Frontend:** React 18, TypeScript, Vite, Zustand (за стейт мениджмънт)
*   **Styling:** Tailwind CSS, Framer Motion
*   **Mobile / Native:** Capacitor, `@capacitor/app`, `cordova-plugin-playlist` (ExoPlayer)

## 🚀 Стартиране на проекта

### Уеб (Локална разработка)
1. Инсталирайте зависимостите: `npm install`
2. Стартирайте сървъра: `npm run dev`

### Компилиране за Android
Проектът разполага с напълно конфигурирана папка `android/`. За да синхронизирате последните промени от уеб кода към Android:

1. Билд на уеб версията: `npm run build`
2. Синхронизиране с Capacitor: `npx cap sync android`
3. Стартиране на Android Studio: `npx cap open android` или компилиране директно чрез `./gradlew assembleDebug` в папка `android/`.

---
*Проектът е автоматизиран чрез GitHub Actions. При всяко качване (Push) в `main` клона, GitHub автоматично генерира нов `.apk` файл!*
